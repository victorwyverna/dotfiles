# dotfiles

This repository contains my dotfiles.

## Requirements

- macOS
- [Homebrew](https://brew.sh/)
- A licensed [Essential PragmataPro](https://fsd.it/shop/fonts/pragmatapro/) font

## Installation

Clone the repository with its submodules and run the installer:

```sh
git clone --recurse-submodules https://github.com/victorwyverna/dotfiles.git
cd dotfiles
./install
```

If the repository was cloned without submodules, `./install` initializes them
automatically.
